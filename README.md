# Swarm Heaven

A Bevy game built with Rust.

## Prerequisites

### Linux

You need to install the following dependencies:

```bash
sudo apt-get update && sudo apt-get install -y libasound2-dev libudev-dev pkg-config
```

### Windows / macOS

No additional system dependencies are usually required beyond the Rust toolchain.

## Running the game

```bash
cargo run
```
